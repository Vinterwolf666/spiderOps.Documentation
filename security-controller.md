# `SecurityController`

**Base route:** `/Customer.Security.Microservice.Customers.API/Security`

This controller provides endpoints to retrieve security reports related to dependencies and secrets, leveraging tools such as GitHub Actions, Gitleaks, and Snyk.

---

## `GET /dependencias?repoUrl=xxx`

Returns the dependency vulnerability report generated by GitHub Actions (e.g., Snyk or .NET CLI).

**Query Parameters:**
- `repoUrl` (string): URL of the GitHub repository.

**Response:**
- `200 OK`: Text report of vulnerable dependencies.
- `400 Bad Request`: If no URL is provided.

---

## `GET /gitleaks?repoUrl=xxx`

Returns the secret scanning report generated by Gitleaks (in SARIF format).

**Query Parameters:**
- `repoUrl` (string): URL of the GitHub repository.

**Response:**
- `200 OK`: SARIF-format secret scanning report.
- `400 Bad Request`: If no URL is provided.
